# Сайт-портфолио с фотогалереей и админкой

Яркий сайт-портфолио художницы Кристины Харламовой: живопись, графика, художественная ковка, керамика. Галерея с фильтрами по категориям, раздел «Обо мне», админка. Без зависимостей — нужен только Node.js.

## Запуск

1. Установите Node.js (v16 или новее): https://nodejs.org
2. В папке проекта выполните:

```
node server.js
```

3. Откройте в браузере:
   - Сайт: http://localhost:3000
   - Админка: http://localhost:3000/admin

## Админка

Пароль по умолчанию: **admin123** — обязательно смените его в разделе «⚙️ Настройки сайта» после первого входа.

В админке можно:

- загружать фото работ (перетаскиванием, сразу несколько) с выбором категории;
- удалять фото;
- добавлять и менять подписи;
- менять имя на главной, подзаголовок и пароль.

## Где хранятся данные

- `uploads/` — сами фотографии;
- `data/photos.json` — список фото и подписи;
- `data/config.json` — заголовок, подзаголовок и пароль.

Для резервной копии достаточно сохранить папки `uploads/` и `data/`.

## Размещение в интернете

Сайту нужен Node.js-хостинг (это не статический сайт, т.к. есть загрузка фото).
Подойдут: VPS (любой), Railway, Render, Fly.io, Glitch. На сервере выполните
`node server.js` — порт можно задать переменной окружения `PORT`.

Если сайт будет доступен из интернета, поставьте перед ним HTTPS
(например, через Caddy или nginx + certbot), чтобы пароль не передавался открыто.

## Содержимое галереи

В галерее — 48 работ из портфолио 2026 года, с подписями (название, год,
материал, размер) и категориями: Живопись, Графика, Ковка, Керамика.
